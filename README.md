Online Shop AddOn für REDAXO 4
============================

Dieses Addon bietet die Möglichkeit ein sehr einfaches Online-Shop
zu betreiben.


Features im Backend
-------------------
* Artikelverwaltung
	* Einträge erfassen / ändern / löschen
* Kategorieverwaltung
	* Einträge erfassen / ändern / löschen
* Kundenverwaltung
	* Einträge erfassen / ändern / löschen
* Bestellverwaltung
	* Einträge einsehen



Hinweise
--------

* Getestet mit REDAXO 4.6
* Frontend-Implementierung folgt demnächst
