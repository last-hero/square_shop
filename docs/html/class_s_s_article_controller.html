protected

Initialisierung.

Erstellen der benötigten Objekte. Dazu zählen Artikel, Artikel-View und ein Array für die Artikelliste. Zudem wird ein ArtikelID gesetzt, Falls die Detailansicht von einer Artikel erwünscht wird.

invoke ( )

Starter.

Artikel Detail/List-Ansicht starten. Falls ArtikelID gesetzt wurde, dann Daten von dieser Artikel aus DB holen Falls kein ArtikelID sondern Kategorie ID gesetzt wurde, dann alle Artikel der Kategorie holen
